📌 About Hirono

Hirono is a **text-based task management application** designed to help you stay **organized and productive**.

  • ✅ **Easy to use**
  • ⚡ **Fast & Efficient**
  • 🔧 **Customizable for your needs**

🚀 Features

  • 📌 **Task Management** - Add, delete, mark, and unmark tasks.
  • 📅 **Event Scheduling** - Manage deadlines and events with time tracking.
  • 🔎 **Search & Filter** - Quickly find tasks using keywords or dates.

💡 Getting Started

Run the application using:

java -jar hirono.jar

Add your first task:

todo read book

Explore the features and let Hirono organize your life!

📌 Task Management Commands

📝 Adding Tasks

Add a new task:

todo [task name]
deadline [task name] /by [yyyy-MM-dd HHmm]
event [event name] /from [yyyy-MM-dd HHmm] /to [yyyy-MM-dd HHmm]

✅ Marking a Task as Done

Mark a task as completed:

mark [task number]

🔄 Unmarking a Task

Unmark a task to mark it as incomplete:

unmark [task number]

🗑 Deleting a Task

Remove a task from the list:

delete [task number]

🔍 Finding Tasks

Search for tasks containing a keyword:

find [keyword]

📅 Filtering Tasks Due by Date

List all tasks due on a specific date:

date [date]
